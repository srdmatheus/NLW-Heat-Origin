## Projeto
Este crachá digital foi criado durante o evento NLW/Heat disponibilizado pela @rocketseat-education. Fui guiado durante 5 aulas pelos professores @maykbrito e @jakeliny. Ao final, tive a oportunidade de inserir funcionalidades extras ao projeto.

<br>

## Funcionalidades extras
Fui além e, adicionalmente, criei um modal para a inserção dos dados de quem for receber o crachá. Também estilizei com efeito acrílico.
